Installare moduli:
```
pip install websockets
pip install pymongo
pip install pandas
pip install paho-mqtt
pip install amqtt
pip install hbmqtt
pip install ujson
pip install networkx
```

Start broker
```
py broker.py
```
Start node(s)
```
py node.py [id] [numberOfNodes]
```
Start master
```
py master.py
```
